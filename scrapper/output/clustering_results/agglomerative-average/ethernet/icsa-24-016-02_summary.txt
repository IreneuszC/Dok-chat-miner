A vulnerability has been identified in the OPC UA Server Toolkit, with a base score of 5.3. Integration Objects has released a security fix to mitigate the risk, available for demo and full versions of the toolkit. CISA recommends defensive measures to minimize the risk of exploitation and encourages organizations to implement cybersecurity strategies. No known public exploitation targeting this vulnerability has been reported. The vulnerability could allow a remote attacker to add content to the log file.
