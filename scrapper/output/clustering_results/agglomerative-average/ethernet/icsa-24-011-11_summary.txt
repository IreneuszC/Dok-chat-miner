Multiple vulnerabilities have been identified in Siemens products that could allow an attacker to execute code in the context of the current process by using specially crafted PAR files. The vulnerabilities include out of bounds read and write, stack overflow, and uninitialized pointer access. The CVSS v3 base score for these vulnerabilities is 7.8. Siemens has provided specific workarounds and mitigations to reduce the risk, and CISA recommends implementing cybersecurity strategies and following recommended practices to protect against these vulnerabilities. No known public exploitation targeting these vulnerabilities has been reported, and they are not exploitable remotely.
