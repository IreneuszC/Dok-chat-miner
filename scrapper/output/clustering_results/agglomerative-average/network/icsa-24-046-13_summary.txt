Siemens has reported vulnerabilities to CISA and released new versions for affected products. The vulnerabilities could allow an attacker to crash the application, execute code, or perform remote code execution. Siemens has provided specific workarounds and mitigations, and recommends protecting network access and configuring the environment according to policy. No known public exploitation has been reported, and organizations should follow established procedures and report any suspicious activity to CISA. Additional mitigation guidance and recommended practices are available on the ICS webpage. For more information, organizations can refer to Siemens security advisory SSA-797296 and industrial security information on the Siemens website.
