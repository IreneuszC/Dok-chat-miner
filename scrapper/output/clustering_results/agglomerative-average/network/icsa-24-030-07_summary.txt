Multiple vulnerabilities have been identified in Rockwell Automation products, with base scores ranging from 6.5 to 8.8. These vulnerabilities can be exploited by authenticated attackers using specifically crafted communication requests to cause denial-of-service conditions, memory overwriting, or remote code execution. Rockwell Automation has recommended risk mitigations and encourages users to apply them. CISA has been informed of these vulnerabilities and advises organizations to implement cybersecurity strategies to protect against potential exploitation. No known public exploitation targeting these vulnerabilities has been reported, but organizations should report any suspected malicious activity to CISA. Additional mitigation guidance and recommended practices are available on the ICS webpage.
