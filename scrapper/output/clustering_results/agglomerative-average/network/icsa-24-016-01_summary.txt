The ICS webpage provides additional mitigation guidance and recommended practices for a vulnerability in MOVITOOLS MotionStudio. No known public exploitation targeting the vulnerability has been reported to CISA, and it is not exploitable remotely. Organizations should report any suspected malicious activity to CISA and implement recommended cybersecurity strategies. The vulnerability has been assigned a CVSS v3.1 base score of 5.5 and can result in unrestricted file access when processing XML information. SEW-EURODRIVE is working to resolve the issue, and an advisory update will be released. Users of affected versions of MOVITOOLS MotionStudio are advised to contact the company for additional information.
