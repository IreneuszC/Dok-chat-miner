Mitsubishi Electric has reported an information disclosure vulnerability in their MELSEC iQ-R Series Safety CPU and SIL2 Process CPU modules. This vulnerability could allow a non-administrator user to access the credentials of a user with lower access levels. A CVSS v3.1 base score of 6.5 has been calculated for this vulnerability. Mitsubishi Electric recommends updating the CPU module and enabling enhanced version of vulnerability management in GX Works3 to prevent the attack. CISA recommends implementing cybersecurity strategies and following established procedures to minimize the risk of exploitation. No known public exploitation targeting this vulnerability has been reported at this time.
