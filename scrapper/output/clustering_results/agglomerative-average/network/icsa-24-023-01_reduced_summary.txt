The APsystems ENERGY COMMUNICATION UNIT (ECU-C) Power Control Software has vulnerabilities that allow attackers to access sensitive data and execute commands with full admin rights. Organizations should report any suspicious activity to CISA and implement recommended cybersecurity strategies to minimize the risk of exploitation. APsystems has not responded to requests to mitigate the vulnerability, and a public proof of concept has been discovered. The CVSS v3.1 base score for this vulnerability is 8.8.
