AVEVA PI Server versions 2023 and 2018 SP3 P05 and prior have vulnerabilities that could lead to denial-of-service conditions. To minimize the risk of exploitation, organizations should apply security updates as soon as possible and implement recommended cybersecurity strategies. Upgrading to AVEVA PI Server version 2023 Patch 1 or later will fix the vulnerabilities. No known public exploitation targeting these vulnerabilities has been reported at this time.
