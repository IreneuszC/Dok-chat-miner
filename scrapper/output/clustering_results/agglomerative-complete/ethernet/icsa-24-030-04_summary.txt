CISA recommends users take defensive measures to minimize the risk of exploitation of vulnerabilities in Hitron Systems DVR. They encourage organizations to implement recommended cybersecurity strategies and report any suspected malicious activity to CISA. Hitron Systems has released updates for their DVR to address the vulnerabilities. A CVSS v3.1 base score of 7.4 has been calculated for the vulnerabilities, which could allow an attacker to cause a denial-of-service condition when using default admin name and password. Multiple versions of Hitron Systems DVR are affected by these vulnerabilities.
