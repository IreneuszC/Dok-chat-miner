Multiple vulnerabilities have been identified in Siemens products, allowing attackers to execute code in the context of the current process by using specially crafted PAR files. The vulnerabilities include out of bounds read and write, stack overflow, heap-based buffer overflow, and uninitialized pointer access. The CVSS v3 base score for each vulnerability is 7.8. No known public exploitation targeting these vulnerabilities has been reported, and they are not exploitable remotely. Siemens has provided workarounds and mitigations, and CISA recommends implementing cybersecurity strategies and following additional mitigation guidance.
