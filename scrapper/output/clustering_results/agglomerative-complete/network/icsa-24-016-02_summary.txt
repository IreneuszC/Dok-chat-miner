A vulnerability has been identified in the OPC UA Server Toolkit, with a base score of 5.3. Integration Objects has released a security fix to mitigate the risk, available for demo version users and full version customers with valid support contracts. CISA recommends defensive measures and cybersecurity strategies to minimize the risk of exploitation. No known public exploitation targeting this vulnerability has been reported. The vulnerability allows a remote attacker to add content to the log file, and affects the OPC UA Server Toolkit OPC library.
