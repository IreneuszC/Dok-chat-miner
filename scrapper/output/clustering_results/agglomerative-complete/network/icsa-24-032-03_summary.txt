CISA recommends users take defensive measures to minimize the risk of exploitation of a vulnerability in AVEVA Edge products, which could allow for arbitrary code execution and privilege escalation. A CVSS v3 base score of 7.3 has been calculated for this vulnerability. AVEVA recommends users upgrade to AVEVA Edge 2023 or AVEVA Edge 2020 R2 SP2 P01 as soon as possible. The vulnerability was discovered and disclosed by Ting Chen of UESTC and ADLab of Venustech. CISA encourages organizations to implement recommended cybersecurity strategies and report any suspected malicious activity. No known public exploitation specifically targeting this vulnerability has been reported at this time.
