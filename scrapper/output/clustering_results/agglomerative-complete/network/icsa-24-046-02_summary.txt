Siemens has identified vulnerabilities in their products, including an infinite loop in OPC UA .NET Standard Stack 1.04.368 and issues with SSL_check_chain() function and X.509 GeneralName type in OpenSSL. These vulnerabilities could allow remote attackers to cause denial-of-service attacks. Siemens has released a new version of SIDIS Prime and recommends users update to the latest version. No known public exploitation has been reported, but organizations are encouraged to implement recommended cybersecurity strategies.
