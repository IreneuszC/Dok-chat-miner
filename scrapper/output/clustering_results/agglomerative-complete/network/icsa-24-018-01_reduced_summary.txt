AVEVA PI Server versions 2023 and 2018 SP3 P05 and prior contain vulnerabilities that could lead to denial-of-service conditions. To minimize the risk of exploitation, users should apply security updates as soon as possible and implement recommended cybersecurity strategies. Organizations should evaluate the impact of these vulnerabilities and report any suspected malicious activity to CISA. Upgrading to AVEVA PI Server version 2023 Patch 1 or later will fix the vulnerabilities.
