Multiple vulnerabilities have been identified in Nice Linear eMerge E3-Series versions 1.00-06 and prior, including cross-site scripting, path traversal, OS command injection, file inclusion, unrestricted upload of malicious files, and buffer overflow. These vulnerabilities could allow attackers to gain unauthorized access, execute arbitrary code, and escalate privileges. Additionally, the system stores passwords in clear-text and contains hard-coded credentials, creating significant security risks. Users are advised to upgrade to the latest firmware and follow recommended cybersecurity strategies to mitigate these vulnerabilities. Organizations should report any suspected malicious activity to CISA for tracking and correlation against other incidents.
