The XPort device server configuration manager is vulnerable to attackers obtaining weakly encoded credentials within web request headers. A CVSS v3.1 base score of 5.7 has been assigned to this vulnerability. Aarn Flecha Menndez of S21Sec reported the vulnerability to CISA. Lantronix recommends upgrading to a product with stronger encryption. CISA advises organizations to implement cybersecurity strategies and follow established procedures for reporting suspicious activity. No known public exploitation targeting this vulnerability has been reported to CISA at this time.
