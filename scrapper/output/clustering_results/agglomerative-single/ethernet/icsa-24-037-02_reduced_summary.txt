The article discusses a vulnerability in HID Global's iCLASS SE reader configuration cards, which could allow attackers to extract sensitive data such as credential and device administrator keys. To mitigate this risk, administrators should securely destroy unneeded configuration cards and update readers and credentials with new keys. HID Global also recommends restricting access to configuration cards and implementing recommended cybersecurity strategies. No known public exploitation of this vulnerability has been reported, and it is not exploitable remotely.
