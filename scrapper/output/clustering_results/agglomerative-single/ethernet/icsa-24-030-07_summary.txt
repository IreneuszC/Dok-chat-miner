Multiple vulnerabilities have been identified in Rockwell Automation products, with base scores ranging from 6.5 to 8.8. These vulnerabilities can be exploited through specifically crafted communication requests, leading to denial-of-service conditions, memory overwriting, or remote code execution. Rockwell Automation has recommended risk mitigations, and CISA encourages organizations to implement cybersecurity strategies to address these vulnerabilities. No known public exploitation targeting these vulnerabilities has been reported, but organizations should report any suspected malicious activity to CISA. Additional mitigation guidance and recommended practices are available on the ICS webpage.
