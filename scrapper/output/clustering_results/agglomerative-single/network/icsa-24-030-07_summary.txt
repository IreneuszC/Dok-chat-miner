Multiple vulnerabilities have been identified in Rockwell Automation products, with base scores ranging from 6.5 to 8.8. These vulnerabilities can be exploited through specifically crafted communication requests, leading to denial-of-service conditions, memory overwriting, or remote code execution. Rockwell Automation has recommended applying risk mitigations and defensive measures to minimize the risk of exploitation. CISA encourages organizations to implement recommended cybersecurity strategies and report any suspected malicious activity. No known public exploitation targeting these vulnerabilities has been reported to CISA at this time.
