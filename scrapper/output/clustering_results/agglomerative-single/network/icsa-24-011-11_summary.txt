Multiple vulnerabilities have been identified in Siemens products, specifically related to parsing specially crafted PAR files. These vulnerabilities could allow an attacker to execute code in the context of the current process. The CVSS v3 base score for each vulnerability is 7.8. No known public exploitation targeting these vulnerabilities has been reported, and they are not exploitable remotely. Siemens has provided specific workarounds and mitigations to reduce the risk, and CISA recommends implementing cybersecurity strategies and following additional mitigation guidance.
