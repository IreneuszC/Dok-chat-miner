SystemK NVR 504/508/516 versions 2.3.5SK.30084998 and prior are vulnerable to a command injection in the DDNS settings, allowing attackers to execute arbitrary commands with root privileges. CISA discovered a public PoC by Keniver Wang and SystemK has not responded to requests to mitigate the vulnerability. Users are advised to contact CISA for more information and take defensive measures. The vulnerability has a CVSS v3.1 base score of 9.8 and organizations should report any suspected malicious activity to CISA. No known public exploitation targeting this vulnerability has been reported. CISA encourages organizations to implement recommended cybersecurity strategies.
