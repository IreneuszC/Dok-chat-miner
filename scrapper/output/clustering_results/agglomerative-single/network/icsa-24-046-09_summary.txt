Siemens has identified multiple vulnerabilities in their products, including issues with email notifications, input field sanitization, OS command handling, and authentication validation. These vulnerabilities could allow attackers to disable notifications, inject code, escalate privileges, and execute arbitrary code on affected devices. Siemens has released new versions for some products and recommends users update to the latest versions. They also provide mitigation guidance and recommend implementing cybersecurity strategies. No known public exploitation of these vulnerabilities has been reported, but organizations are encouraged to report any suspected malicious activity to CISA.
