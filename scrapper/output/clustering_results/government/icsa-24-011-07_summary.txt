CISA recommends organizations to perform impact analysis and risk assessment before deploying defensive measures. Siemens has reported a vulnerability that could allow an authenticated local attacker to inject arbitrary code and gain root access. CISA will no longer update ICS security advisories for Siemens product vulnerabilities after January 10, 2023. No known public exploitation targeting this vulnerability has been reported, and it is not exploitable remotely. CISA encourages organizations to implement recommended cybersecurity strategies and offers additional mitigation guidance and best practices.
