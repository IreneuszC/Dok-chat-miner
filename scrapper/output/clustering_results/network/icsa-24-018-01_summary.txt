AVEVA PI Server versions 2023 and 2018 SP3 P05 and prior contain vulnerabilities that could allow unauthenticated users to cause denial-of-service conditions. AVEVA recommends upgrading to version 2023 Patch 1 or later, or applying security updates for affected products. CISA advises organizations to evaluate the impact of these vulnerabilities and take defensive measures, and provides additional mitigation guidance and best practices for cybersecurity. No known public exploitation targeting these vulnerabilities has been reported, but organizations should report any suspected malicious activity to CISA.
