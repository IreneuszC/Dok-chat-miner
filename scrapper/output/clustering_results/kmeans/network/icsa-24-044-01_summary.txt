Mitsubishi Electric has reported an information disclosure vulnerability in the MELSEC iQ-R Series Safety CPU and SIL2 Process CPU modules, which could allow a non-administrator user to access the credentials of a user with lower access level. A CVSS v3.1 base score of 6.5 has been calculated for this vulnerability. A workaround is available for MELSEC iQ-R Series Safety CPU versions 27 or later when used with GX Works3 versions 1.087R or later. Mitsubishi Electric recommends updating the CPU module and implementing mitigation measures to minimize the risk of exploitation. CISA also recommends defensive measures and provides additional guidance for organizations to address this vulnerability. No known public exploitation targeting this vulnerability has been reported at this time.
