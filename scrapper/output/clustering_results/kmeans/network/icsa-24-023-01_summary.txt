The APsystems ENERGY COMMUNICATION UNIT (ECU-C) Power Control Software V4.1NA V3.11.4 W2.1NA V4.1SAA C1.2.2 has a vulnerability that allows attackers to access sensitive data and execute commands with full admin rights. APsystems has not responded to requests to mitigate this vulnerability, and users are encouraged to take defensive measures. CISA has assigned a CVSS v3.1 base score of 8.8 to this vulnerability and recommends organizations to report any suspected malicious activity. No known public exploitation targeting this vulnerability has been reported to CISA at this time.
