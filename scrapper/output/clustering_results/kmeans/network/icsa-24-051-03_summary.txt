Mitsubishi Electric has reported a vulnerability in Microsoft Message Queuing service that affects their electrical discharge machines. This vulnerability could allow attackers to execute remote code, disclose, tamper with, destroy, or delete information, or cause a denial-of-service condition. No known public exploitation has been reported, but organizations are encouraged to follow cybersecurity strategies and implement recommended mitigations. Mitsubishi Electric recommends installing the latest update and has reported the vulnerability to CISA, which has assigned it a CVSS v3.1 base score of 9.8.
