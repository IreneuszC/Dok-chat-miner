CISA recommends performing impact analysis and risk assessment before deploying defensive measures. Crestron has resolved an OS command injection vulnerability in firmware version 1.4499.00023.001 or higher. The vulnerability was reported by Uri Katz of Claroty Research Team82 and has a CVSS v3.1 base score of 8.4. No known public exploitation targeting this vulnerability has been reported, and it is not exploitable remotely. CISA encourages organizations to implement recommended cybersecurity strategies and report any suspected malicious activity.
