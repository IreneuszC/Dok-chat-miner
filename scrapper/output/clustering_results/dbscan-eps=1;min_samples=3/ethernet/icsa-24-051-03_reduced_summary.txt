Mitsubishi Electric has reported a remote code execution vulnerability in Microsoft Message Queuing service on their electrical discharge machines. Organizations should follow internal procedures and report any suspicious activity to CISA. To minimize the risk of exploitation, users should implement recommended cybersecurity strategies and install the latest update from Mitsubishi Electric. The vulnerability has a high CVSS base score of 9.8 and could allow attackers to tamper with or delete information, or cause a denial-of-service condition.
