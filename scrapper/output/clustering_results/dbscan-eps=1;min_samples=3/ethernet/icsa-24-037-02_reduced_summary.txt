The article discusses a vulnerability in HID Global's iCLASS SE reader configuration cards, which could allow attackers to extract sensitive data such as credential and device administrator keys. To mitigate this risk, HID Global recommends updating readers and credentials with new keys, securely destroying unneeded configuration cards, and restricting access to configuration cards. The vulnerability is not exploitable remotely, but organizations are encouraged to implement recommended cybersecurity strategies and report any suspected malicious activity to CISA.
