Multiple vulnerabilities have been identified in Siemens products, specifically related to parsing specially crafted PAR files. These vulnerabilities could allow an attacker to execute code in the context of the current process. The affected applications have been assigned a CVSS v3 base score of 7.8. Siemens has provided specific workarounds and mitigations to reduce the risk, and CISA recommends implementing cybersecurity strategies and following established internal procedures to protect against these vulnerabilities. No known public exploitation targeting these vulnerabilities has been reported to CISA at this time.
