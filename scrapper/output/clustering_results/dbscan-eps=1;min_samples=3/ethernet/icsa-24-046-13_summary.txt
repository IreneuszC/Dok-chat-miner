Siemens reported vulnerabilities to CISA and released updates for affected products. The vulnerabilities could allow attackers to crash applications, execute code, and perform remote code execution. Siemens recommends updating to the latest versions and following specific workarounds and mitigations. No known public exploitation targeting these vulnerabilities has been reported, and organizations should report any suspected malicious activity to CISA. Additional mitigation guidance and recommended practices are available on the ICS webpage and in product manuals. For more information, see Siemens security advisory SSA-797296 and visit the Siemens industrial security website.
