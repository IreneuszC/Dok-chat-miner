AVEVA PI Server versions 2023 and 2018 SP3 P05 and prior contain vulnerabilities that could allow an attacker to crash the product or throttle memory, leading to a partial denial-of-service condition. AVEVA recommends applying security updates and implementing defensive measures. CISA recommends organizations to evaluate the impact of these vulnerabilities and implement recommended cybersecurity strategies. No known public exploitation targeting these vulnerabilities has been reported. All affected versions can be fixed by upgrading to AVEVA PI Server version 2023 Patch 1 or later.
