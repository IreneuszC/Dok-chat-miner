Multiple vulnerabilities have been identified in the Nice Linear eMerge E3-Series versions 1.00-06 and prior, including cross-site scripting, path traversal, OS command injection, file inclusion, unrestricted upload of malicious files, SSH with hardcoded credentials, buffer overflow, plaintext password storage, authorization mechanism vulnerability, and hard-coded credentials in binaries. These vulnerabilities could allow an attacker to gain unauthorized access, execute arbitrary code, escalate privileges, and obtain sensitive information. Users are advised to upgrade to the latest firmware and follow recommended cybersecurity strategies. Organizations should report any suspected malicious activity to CISA.
