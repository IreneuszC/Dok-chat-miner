The XPort device server configuration manager is affected by a vulnerability that allows attackers to obtain weakly encoded credentials. A CVSS v3.1 base score of 5.7 has been calculated for this vulnerability. Aarn Flecha Menndez of S21Sec reported the vulnerability to CISA. Lantronix recommends upgrading to a product with stronger encryption. CISA advises organizations to implement cybersecurity strategies and follow established procedures for reporting suspicious activity. No known public exploitation of this vulnerability has been reported at this time.
