Emerson Rosemount GC370XA, GC700XA, and GC1500XA products have multiple vulnerabilities that could allow an unauthenticated user with network access to obtain sensitive information, cause a denial-of-service condition, and bypass authentication to acquire admin capabilities. Users are advised to update the affected products' firmware and implement cybersecurity best practices. No known public exploitation targeting these vulnerabilities has been reported, but organizations should report any suspected malicious activity to CISA. CVSS v3.1 base scores have been calculated for the vulnerabilities, ranging from 6.9 to 9.8.
