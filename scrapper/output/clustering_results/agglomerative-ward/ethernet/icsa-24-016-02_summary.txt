A vulnerability with a CVSS v3.1 base score of 5.3 has been reported by Sam Hanson of Dragos to CISA, allowing a remote attacker to add content to the log file. Integration Objects has released a security fix for the OPC UA Server Toolkit to mitigate the risk. CISA recommends defensive measures and cybersecurity strategies to minimize the risk of exploitation. No known public exploitation targeting this vulnerability has been reported at this time.
