Mitsubishi Electric has reported a vulnerability in their MELSEC iQ-R Series Safety CPU and SIL2 Process CPU modules, which could allow a non-administrator user to disclose the credentials of a user with a lower access level. A CVSS v3.1 base score of 6.5 has been calculated for this vulnerability. Mitsubishi Electric recommends enabling enhanced version of vulnerability management of GX Works3 when writing user information to the CPU module to prevent the attack. Users are advised to contact their local Mitsubishi Electric representative for updates and to implement recommended cybersecurity strategies. No known public exploitation targeting this vulnerability has been reported at this time.
