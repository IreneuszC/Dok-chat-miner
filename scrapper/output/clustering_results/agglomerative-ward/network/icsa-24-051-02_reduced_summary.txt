Industrial Control Systems Network Protocol Parsers (ICSNPP) - Ethercat Zeek Plugin versions d78dda6 and prior are vulnerable to out-of-bounds read and write, which could allow an attacker to crash the system or execute arbitrary code. To protect against these vulnerabilities, users are advised to update the plugin and keep critical software updates and patches up to date. Additionally, organizations should implement recommended cybersecurity strategies and report any suspected malicious activity to CISA for tracking and correlation against other incidents.
